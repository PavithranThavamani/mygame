# Deploy Your Memory Game

## Fix Dependencies First:
```
npm install --legacy-peer-deps
npm run build
```

## Quick Deployment Steps:

### Option 1: Netlify (Easiest)
1. Run the commands above
2. Go to netlify.com and sign up
3. Drag the `build` folder to Netlify
4. Get your live link instantly

### Option 2: Vercel
1. Run the commands above
2. Go to vercel.com and sign up  
3. Import your project or drag the `build` folder
4. Deploy and share the link
